No tienes dispositivos aún

No tienes acceso a ese dispositivo. Puedes asignar (claim) uno a tu cuenta.

Para ver tu panel, primero tienes que asignar (claim) un ESP32 a tu cuenta.

Necesitas el código del dispositivo (device_code) y el token del dispositivo (device_token) (es el mismo que usa el ESP32 como X-Device-Token).

¿Dónde están estos datos?
1) Abre tu firmware ESPHome (archivo riego_esp32.yaml).
2) Arriba verás algo como:
device_code: RIEGO_001
device_token: (un texto largo)
3) El device_token no lo “da” la web: lo defines tú (instalación). Debe tener 12+ caracteres y no puede ser CAMBIA_ESTE_TOKEN.
4) Flashea el ESP32 con ese firmware. Luego pega aquí ese mismo token.
